# 🎬 ShowSphere

A modern and responsive movie ticket booking web application inspired by real-world cinema booking platforms. ShowSphere provides an engaging user experience with movie listings, seat booking, food ordering, booking confirmation, and a clean cinematic interface.

---

## 🌐 Live Demo

🔗 **Website:** https://tanishqkamboj5-rgb.github.io/ShowSphere/

---

## ✨ Features

- 🎥 Modern movie browsing interface
- 🎟️ Interactive ticket booking
- 🍿 Food & beverages ordering section
- 🪑 Seat selection interface
- ✅ Booking confirmation page
- 🏙️ City selection support
- 📱 Responsive user interface
- 🎨 Premium cinema-inspired design

---

2. Open the project folder

3. Launch `index.html` in your browser
